Poio

Technologies for Language Diversity

Poio is a free and open source predictive text and transliteration system for under-resourced languages. We support text input for languages like Bavarian, Chechen and Sorbian on mobile phones and desktop computers.

Read more about the project on readthedocs:

http://poio.readthedocs.org
